# Aserto Swagger-UI 

Aserto-themed Swagger-UI React component

## Installation

Yarn: `yarn install @aserto/aserto-swagger-ui` 

NPM: `npm install @aserto/aserto-swagger-ui`

## Usage

```javascript
import AsertoSwaggerUI from '@aserto/aserto-swagger-ui'

const View = ({url}) => 
  <AsertoSwaggerUI url={url} />

export default View
```
